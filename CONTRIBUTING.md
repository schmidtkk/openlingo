# Bugs

Please report bugs in Issues, we have a background agent that fixes them semi-autonomously.

# Features

Best way to get your request implemented is to describe your request in great detail in Issues. You can also open a PR with your detailed feature description inside `task.md` at the root level. If we like it we will implement it on top of your commit.

# Contributing code

AI Slop PRs with hundreds of lines and AI generated descriptions will be automatically closed.

You can use AI to assist your coding but the description of the PR and what to look for should be human writen (we don't care if your English is not perfect). You will also need to provide screenshots and videos in the PR.
